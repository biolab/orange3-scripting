Orange 3 Scripting Installation

This step by step guide helps you install the scripting part of Orange 3 and make it work with the IPython notebook.

Software Installation Instructions

Step 1: Python data mining libraries

For Mac OS X, Windows, or Linux.

Step 2: Testing the Environment

Download Testing Python Environment Notebook and open it with IPython notebook.

Step 1: Mac OS X

Automatic setup

Install can be automated by using Mac OS X setup script setup-mac.sh: please download the script file and save it to directory where you would like to place your software installation. Setup will install Orange3, numpy and scikit-learn and their dependencies in a virtual environment, and finally run IPython notebook. Before the installation, open the terminal and move to directory, where you want Orange3 to be installed. The following command wil start the installation, which should take about 15 minutes. 

 source setup-mac.sh

Install script will also create and enter new virtual environment, which you can reenter using following command:

 source orange3env/bin/activate

The installed packages can be used only from within virtual environment.

Manual installation

Instead of automatic install, you can install the software manually. Open a terminal and run the following commands:

Install homebrew package manager by executing (it will ask you to install XCode Command Line Tools): 
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Enable science packages and install dependencies 

brew tap homebrew/science

Install the necessary system packages:

brew install makedepend git python3 gcc suite-sparse swig

We will use Python3 and need to create a virtual environment for it:

pip3 install virtualenv
virtualenv -p python3 orange3env
source orange3env/bin/activate

Install Orange and its requirements (this step will take about 15 minutes): 

git clone https://github.com/biolab/orange3
cd orange3
pip install numpy
pip install -r requirements.txt
python setup.py develop
pip install ipython[all] Orange-Bioinformatics networkx python-igraph matplotlib

Now you can start IPython environment and import Orange. To exit ipython type Ctrl+D. 

ipython
>>> import Orange

To start IPython notebook, write the following line 

ipython notebook

If you get ValueError: unknown locale: UTF-8, you can solve it using following commands: 

export LC_ALL=en_US.UTF-8
export LANG=en_US.UTF-8

Step 1: Windows

In a nutshell

  • Download and open Orange3-3.2.dev-current.win32-py3.4-install.exe from http://orange.biolab.si/download/files.
  • Follow installation wizard to install Orange3 and its dependencies (Python3.4, NumPy, ...).
  • Open windows command prompt and start IPython notebook with command 'C:\Python34\Scripts\ipython3.exe notebook'.

Detailed installation instructions

Download the installation file Orange3-3.2.dev-current.win32-py3.4-install.exe Save the downloaded file to an arbitrary location. Run the installation by double clicking the downloaded file. Choose Yes when asked whether you want to allow this program to make changes to your computer.

Setup will ask you to confirm installation of Python3.4 on your computer. Click OK. If you already have Python 3.4 installed, this step is not needed.

Follow installation dialog to complete installation of Python3.4. You can leave default settings and wait for installation to complete.

After installation of Python3.4, wizard will install Orange3 and its dependencies. This step might take a while.

Go to windows start menu and write 'cmd'. Program cmd will appear on list as shown on picture below. Click on it and a window will open.

If you used default settings during installation, you can enter python environment by writing following line: 

 C:\Python34\python.exe

To start IPython notebook write the following line: 

 C:\Python34\Scripts\ipython3.exe notebook

Step 1: Linux

Ubuntu (tested on Ubuntu 12.04 and 14.04)

Automatic setup

Install can be automated by using Ubuntu setup script . Setup will install Orange3 and its dependencies in a virtual environment, and finally run IPython notebook. Before the installation, move to directory, where you want Orange3 to be installed. Following command wil start installation, which usually takes arround 15 minutes. 

 source setup-ubuntu.sh

Install script will also create and enter new virtual environment for you, which you can reenter using following command:

 source orange3env/bin/activate

Orange can be used only from within virtual environment.

Manual installation

Alternatively, you can choose manual installation. Open a terminal and run the following commands.

Install the necessary system packages:

sudo apt-get update
sudo apt-get install git python-pip python-virtualenv 
	python3-dev python3-numpy python3-scipy 
	python3-matplotlib ipython3-notebook 
	libxml2-dev zlib1g-dev

We will use Python3 and need to create a virtual environment for it:

mkdir orange3env
virtualenv -p python3 --system-site-packages orange3env
source orange3env/bin/activate

Install Orange and its requirements (this step will take about 15 minutes): 

git clone https://github.com/biolab/orange3
cd orange3
pip install -r requirements.txt
python setup.py develop
pip install Orange-Bioinformatics networkx python-igraph

Now you can check if Orange was successfully installed from IPython environment. To exit ipython type Ctrl+D. 

ipython3
>>> import Orange

To start IPython notebook, write the following line 

ipython3 notebook

Step 2: Testing the Environment

Once you successfully installed the relevant libraries as instructed above please download Testing Python Environment Notebook. If you use Safari, you need to change file extension to .ipynb, otherwise the file will not be recognized by IPython Notebook.

Start IPython notebook from the directory with the downloaded file. Command that starts IPython notebook for Mac OS X is 

ipython notebook

for Linux 

ipython3 notebook

and for Windows 

C:\Python34\Scripts\ipython3.exe notebook

Then open Testing Python Environment Notebook from the IPython Notebook Dashboard. Follow the instructions therein.

That's it - Enjoy Orange.
